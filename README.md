# OpenClaw — Local Setup

Single local OpenClaw instance. One setup, one place: `~/Desktop/openclaw`.

## Quick Start

```bash
cd ~/Desktop/openclaw
docker compose up -d
```

Open: **http://localhost:48728**

No login required — auto-authenticates locally.

## Stop / Restart

```bash
docker compose down       # stop
docker compose restart    # restart
docker compose logs -f    # follow logs
```

## Adding API Keys

Edit `docker-compose.yml` under `environment`:

```yaml
- OPENAI_API_KEY=sk-...
- ANTHROPIC_API_KEY=sk-ant-...
- GEMINI_API_KEY=...
- XAI_API_KEY=...
```

Then restart: `docker compose restart`

## Changing Default Model

Edit `data/openclaw.json`:

```json
"agents": {
  "defaults": {
    "model": {
      "primary": "Gemini 3 Flash Preview"
    }
  }
}
```

Then restart: `docker compose restart`
